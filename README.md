## 安装教程

### 1、导入代码

直接将`poem`文件夹导入到微信小程序的开发工具上



## 2、导入数据

将`data`文件夹下的所有`json`文件，都导入微信小程序的云数据库上

自己找图片导入到微信小程序的云存储上，像这样：
